.. topic:: Abstract

   This document describes the *OpenSHMEM for Python* package.
   *OpenSHMEM for Python* provides Python bindings for the *OpenSHMEM*
   standard, allowing Python applications to exploit multiple
   processors on workstations, clusters, and supercomputers using a
   *Partitioned Global Address Space* (PGAS) programming model.
